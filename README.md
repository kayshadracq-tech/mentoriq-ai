# MentorIQ AI 🤖

MentorIQ AI is a free online AI chatbot that acts as a digital learning mentor.

It helps users:
- Ask questions
- Learn new skills
- Get instant AI responses
- Use a simple web interface

## 🌐 Features
- Free AI chatbot
- Web-based access
- Simple and fast UI
- Educational assistant

## 🚀 Project Status
Currently in development.

## 👨‍💻 Built With
- JavaScript
- Node.js
- Express
- Open-source AI model
